# Residence Page Assets

This folder contains all images and downloadable assets for the `/residence` page.

## Folder Structure

```
public/residence/
├── architecture/       # Architectural exterior shots
├── interior/          # Interior design photos
├── outdoor/           # Outdoor living spaces
├── wellness/          # Wellness & entertainment areas
├── floorplans/        # Floor plan images
│   ├── level1.jpg    # Level 1 dimensioned plan
│   └── level2.jpg    # Level 2 dimensioned plan
├── gallery/           # Gallery preview images
└── downloads/         # PDF downloads
    ├── floorplans.pdf
    ├── materials.pdf
    ├── specifications.pdf
    └── construction-set.pdf
```

## Image Requirements

### Recommended Sizes
- **Hero images**: 1920x1080px (landscape)
- **Section images**: 1200x800px (landscape)
- **Gallery thumbnails**: 800x800px (square)
- **Floor plans**: High resolution (2400px width minimum)

### Format
- Use `.jpg` for photographs
- Use `.png` for floor plans with transparency needs
- Optimize all images for web (use tools like TinyPNG or ImageOptim)

### Naming Convention
- Use lowercase
- Use hyphens for spaces
- Be descriptive: `exterior-front-view.jpg`, `master-bedroom.jpg`

## Downloads

Place PDF files in the `downloads/` folder:
- `floorplans.pdf` - Complete dimensioned floor plans
- `materials.pdf` - Material selections and finish board
- `specifications.pdf` - Technical specifications
- `construction-set.pdf` - Sanitized construction drawings
