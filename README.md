# Ascaya Residence - 11 Stoneshead Ct

Ultra-luxury modern residence website showcasing 11 Stoneshead Ct, Henderson, Nevada.

## Built With

- Next.js 14 (App Router)
- Tailwind CSS
- Framer Motion
- TypeScript

## Development

```bash
# Install dependencies
pnpm install

# Run development server
pnpm dev

# Build for production
pnpm build

# Start production server
pnpm start
```

## Features

- Cinematic hero section with video background
- Interactive image gallery with lightbox
- Instagram integration with Zarios Construction
- Smooth animations and transitions
- Fully responsive design
- Contact form for tour inquiries
